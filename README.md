## Dataset Description
**Dataset Description**

The dataset used for this analysis is the **2023 Yellow Taxi Trip Data**, obtained from the New York City Taxi and Limousine Commission through Data.gov. This dataset contains detailed information on taxi trips taken in New York City during 2023.

Each row in the dataset represents **a single taxi trip**, meaning every observation corresponds to one completed ride. The dataset is large, consisting of **millions of rows (trip records)** and approximately **15–20 columns (variables)**, depending on the specific file used.

The dataset includes several key types of variables that describe each trip:

* **Date/Time Variables**

  * *tpep_pickup_datetime* – the date and time when the trip began
  * *tpep_dropoff_datetime* – the date and time when the trip ended

* **Location Variables**

  * *PULocationID* – the taxi zone where the trip started
  * *DOLocationID* – the taxi zone where the trip ended

* **Trip Characteristics**

  * *trip_distance* – the total distance of the trip in miles
  * *passenger_count* – the number of passengers reported by the driver

* **Fare and Payment Information**

  * Itemized fare variables such as *fare_amount*, *tip_amount*, and *total_amount*, which capture the cost breakdown of each trip
  * *payment_type* – indicates how the passenger paid (e.g., cash or credit card)
  * *rate_type* – indicates the rate applied to the trip (e.g., standard rate, special rates)

The dataset contains a mix of **data types**, including:

* **Numerical variables** (e.g., trip distance, fare amounts, passenger count)
* **Categorical variables** (e.g., payment type, rate type, location IDs)
* **Date/time variables** (e.g., pickup and drop-off timestamps)

Overall, this dataset provides a comprehensive view of **taxi activity, pricing, and rider behavior** in New York City. Its structured, tabular format allows for analysis of patterns in travel, cost, and customer behavior, making it suitable for answering questions related to transportation trends and economic factors.
